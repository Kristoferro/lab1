Knapsack Problem Solver (.NET 8.0)
Projekt realizujący rozwiązanie problemu plecakowego za pomocą algorytmu zachłannego.
Funkcje:
- Tworzenie przedmiotów o losowej wadze i wartości na podstawie podanego przez uzytkownika ziarna (seed).
- Algorytm zachłanny sortowanie przedmiotów według stosunku wartości do wagi i mozliwie jak najszybsze niekoniecznie optymalne wypelnienie plecaka.
- Testy jednostkowe automatyczna weryfikacja poprawności algorytmu i generatora.

Techniczne:
- Jezyk: C#
- Platforma: .NET 8.0 (LTS)
- Srodkowisko: Visual studio 2022
- Framework testowy: MSTest
